Terminal Application

Brief Build an application in Ruby, to be run in the command line.

Tech StackRuby 2.5.1 , Five Ruby Gems (including TTY-prompt and rubocop), Github

Time Frame Three days.

My Aim To build an interactive application that highlights my understanding of scope, classes, methods and error handling.

My Approach I began with ideating and drafting three potential applications. I settled on the game of blackjack as I wanted the challenge of not only programming a bot dealer but testing my algorithmic thinking and problem solving abilities that are needed to formulate the game of blackjack.

Personal Online Portfolio

Brief Design and develop a static website to become the basis of an online portfolio for future employers.

Tech StackHTML5, CSS, deployed through Netlify, Github

My AimTo design an online portfolio that had a minimal aesthetic, conveyed my character and exemplified the potential future company I would like to be part of.

My ApproachThe process started with gathering inspiration through Dribble and my environment. I settled on a Melbourne theme, as many would say I am "very Melbourne."
