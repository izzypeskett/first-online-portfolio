Terminal Application

Brief Build an application to be run in terminal in Ruby

Time-Frame Three days

My Aim Build an application that is interactive, demonstrates my ability to understand global and local scope, classes and methods and handles user error.

My Approach I began by ideating three potential options. I came up with a game of blackjack, an interactive food ordering menu and a phonebook.
Blackjack was my prefered choice as creating a game would be enjoyable for the user. I liked the challenge of not only the technical components of creating the code but the algorithmic and problem solving part of the game of blackjack.
